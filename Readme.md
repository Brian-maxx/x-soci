# 🚧 Project Status

## Overview

This project is currently under active development and is being designed as a scalable system with future AI integration capabilities. The application focuses on building a stable core architecture before implementing advanced AI training and automation features.

I am responsible for system analysis, database structure planning, and overall backend architecture design. The database flow and system structure documentation can be found in the `flow/` directory of this repository.

# 🗂 System Architecture

The project is designed with a layered architecture approach to improve maintainability and scalability.

Main backend layers include:

* Controller Layer
* Service Layer
* Repository Layer
* Database Layer

Database schemas, system analysis documents, and workflow diagrams are maintained inside the `flow/` directory.

---

# 🐳 Running the Project

This project uses Docker for environment setup and deployment.

## Start the application

```bash
docker compose up --build
```

## Stop the application

```bash
docker compose down
```
